# Command Prompt Index

Prompts are stored under `.codex/prompts/`.

- `/pickup`: Session rehydration checklist before starting work.
- `/handoff`: Structured handoff notes for the next agent.
- `/fix`: End-to-end issue fix workflow with tests and checks.
- `/landpr`: End-to-end PR landing workflow with validation and merge checks.

Use these prompts as repo-local defaults for codex-first automation.
